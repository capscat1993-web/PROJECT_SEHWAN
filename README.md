# PROJECT_SEHWAN

프로젝트 시작용 기본 세팅입니다.

## 시작하기

1. 가상환경 활성화
   - PowerShell: `.\.venv\Scripts\Activate.ps1`
2. 패키지 설치
   - `pip install -r requirements.txt`
3. 개발 시작
   - 소스 코드는 `src/` 디렉토리에 작성 권장

## 기본 명령어

- 비활성화: `deactivate`
- 현재 설치 패키지 저장: `pip freeze > requirements.txt`
